# Krista Infrastructure Python

Python packages and utilities for Krista infrastructure components.

## Packages

### krista-infinispan
Python package with Infinispan integration for caching and data storage.

- **Location:** `krista-infinispan/`
- **Installation:** `pip install krista-infinispan`
- **Documentation:** [krista-infinispan/README.md](krista-infinispan/README.md)

## Development

### Setup
```bash
git clone https://github.com/kristasoft/krista-infra-python.git
cd krista-infra-python
```

### Installing packages for development
```bash
# Install krista-infinispan in development mode
cd krista-infinispan
pip install -e .
```
