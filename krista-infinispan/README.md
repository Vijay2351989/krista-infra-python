# Krista Infinispan

Python package with Infinispan integration for Krista infrastructure.

## Installation

### From PyPI
```bash
pip install krista-infinispan
```

### From GitHub
```bash
pip install git+https://github.com/kristasoft/krista-infra-python.git#subdirectory=krista-infinispan
```

### For Development
```bash
git clone https://github.com/kristasoft/krista-infra-python.git
cd krista-infra-python/krista-infinispan
pip install -e .
```

## Usage

```python
from krista_infinispan import InfinispanClient

client = InfinispanClient()
```
